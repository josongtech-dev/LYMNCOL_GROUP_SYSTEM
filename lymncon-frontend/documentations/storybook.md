# Storybook Documentation Notes

Official docs: https://storybook.js.org/docs

## Quick setup used in this project

1. Initialize Storybook:
   - `npm create storybook@latest -- --yes`
2. Run Storybook:
   - `npm run storybook`
3. Build Storybook:
   - `npm run build-storybook`

## Notes

- Storybook is installed for component reference and UI testing in isolation.
- Configuration files are generated in `.storybook/`.
