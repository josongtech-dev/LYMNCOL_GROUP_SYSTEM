# Vite Documentation Notes

Official docs: https://vite.dev/guide/

## Quick start used in this project

1. Create app:
   - `npx create-vite@latest lymncon-frontend --template react-ts`
2. Install dependencies:
   - `npm install`
3. Run dev server:
   - `npm run dev`
4. Build for production:
   - `npm run build`
5. Preview production build:
   - `npm run preview`

## Notes

- Template used: `react-ts`
- Vite is the frontend build tool and dev server for this project.
