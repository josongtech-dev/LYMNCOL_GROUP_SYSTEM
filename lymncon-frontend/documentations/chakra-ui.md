# Chakra UI Documentation Notes

Official docs: https://chakra-ui.com/docs/get-started/frameworks/vite

## Quick setup used in this project

1. Install dependencies:
   - `npm install @chakra-ui/react @emotion/react`
2. Add Chakra provider in `src/main.tsx`.
3. Use Chakra components in pages, for example:
   - `Box`, `Stack`, `Heading`, `Button`, `Link`, `Text`

## Notes

- This project uses Chakra as the main UI component framework.
- Keep UI implementation simple and aligned with official docs.
